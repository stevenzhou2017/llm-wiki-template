# AI 辅助编程

LLM 驱动的编程辅助，从代码补全到完整的对话式开发。

## 发展光谱

从低介入到高介入：

1. **代码补全**：GitHub Copilot 式的行级建议
2. **对话式编辑**：Cursor Composer，在对话中修改代码
3. **[[Vibe Coding]]**：完全不看代码，只描述意图，全盘接受 LLM 输出
4. **自主 Agent**：Claude Code、Codex，LLM 自主规划和执行多步任务

## 关键工具

- Cursor（+ Composer）
- Claude Code
- GitHub Copilot
- OpenAI Codex

## 争议

[[Vibe Coding]] 引发的讨论：这种方式适合原型和周末项目，但对生产环境代码的质量、安全性、可维护性存在顾虑。
